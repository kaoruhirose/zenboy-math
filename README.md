# 然BOY おもしろ算数

5年生向けの、楽しく算数を学べる iPad / Web アプリ。
キャラクター「然BOY」と一緒に、**九九・倍数・約数・最小公倍数・最大公約数**をゲームで身につけます。

## あそべる内容
- 🧭 **つまずき診断** — 10問・速さも計測して、弱点ゾーンに「スタート」旗を立てる
- ✖️ **九九ダッシュ** — スピード勝負。速いとボーナス、連続正解で🔥
- 🐸 **カエルとび** — n の倍数を全部タップ
- 🍪 **クッキーわけ** — n 個を余りなく配れる人数（約数）を全部見つける
- 🤝 **ふたりのカエル** — 2匹の点が重なる数＝最小公倍数 / 同じ数ずつ分ける最大数＝最大公約数
- 🎉 星でレベルアップ → 然BOYに王冠（進捗は端末に自動保存）

## 使い方
- ブラウザで `index.html` を開くだけ（インストール不要・オフライン対応）。
- iPad の Safari で開き、共有 → **「ホーム画面に追加」** すると、アプリのように全画面で起動できます。

## 技術メモ
- 依存ライブラリなしの単一HTML＋PWA（manifest / service worker）。
- データは `localStorage` に保存。外部送信なし。
